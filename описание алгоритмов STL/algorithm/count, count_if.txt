count() выдает количество элементов контейнера, которые удовлетворяют условию 

первым параметром принимает начало контейнера, вторым - конец контейнера, а 
третьим принимает знчение элемента

таким образом count(vector.begin(), vector.end(), 1) вернет количество единиц
в векторе интов


count_if() же третьим параметром принимает функцию либо лямбду(типо свой метод отбора)

Таким образом 

bool is_even(int a)
{
	if(a % 2 == 0)
	{
          return true;
	}
	else
	{
	  return false;
	}
}

count_if(vector.begin(), vector.end(), is_even()) вернет количсетво 
четных элементов в векторе
